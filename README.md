# Image-Text_AI
# 🖼️ Image to Text Converter

A web application that analyzes uploaded images and generates 
detailed text descriptions using Vision AI.

## 🚀 Live Demo
[Click here to view the app](your-streamlit-url-here)

## 🛠️ Built With
- [Streamlit](https://streamlit.io/) - Web framework
- [Groq API](https://console.groq.com/) - AI inference
- [Llama 4 Scout 17B](https://huggingface.co/meta-llama/Llama-4-Scout) - Vision language model
- [Pillow](https://pillow.readthedocs.io/) - Image processing

## ✨ Features
- Upload any image in JPG, JPEG or PNG format
- AI analyzes and describes the image in detail
- Base64 image encoding for API transmission
- Fast vision AI powered by Llama 4 Scout 17B
- Clean and simple user interface
